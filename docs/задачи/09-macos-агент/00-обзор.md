# Портирование агента на macOS — Обзор

## Цель

Реализовать macOS-агент (Swift 5.10 / ScreenCaptureKit) с полным паритетом функциональности Windows-агента (C# .NET 8), используя те же серверные API.

- **URL:** `https://services-test.shepaland.ru/screenrecorder`
- **Учётная запись для проверки:** `maksim` / `#6TY0N0d` (tenant "Тест 1")
- **Регистрационный токен:** `drt_702c` (tenant "Тест 1", max_uses: 10)

## Общие конвенции API

- Все эндпоинты: JSON, `snake_case`
- Timestamps: ISO 8601 (RFC 3339)
- ID: UUID v4
- Аутентификация: `Authorization: Bearer <jwt>`, `X-Device-ID: <device_id>`
- Ошибки: `{"error": "message", "code": "ERROR_CODE"}`

## HTTP-коды и реакция агента

| Код | Значение | Реакция |
|-----|----------|---------|
| 200/201/204 | Успех | Продолжить |
| 400 | Невалидный запрос | Лог, отбросить |
| 401 | Токен истёк | device-refresh → retry |
| 404 | Не найдено | Лог, отбросить |
| 409 | Конфликт (сессия) | Закрыть старую → retry |
| 503/504 | Сервер недоступен | Retry exponential backoff (3 попытки, 2^n сек) |
| 500+ | Ошибка сервера | Лог, retry с backoff |
