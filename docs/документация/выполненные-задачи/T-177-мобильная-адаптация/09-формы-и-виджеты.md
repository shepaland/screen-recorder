# T-177.9: Формы, Dashboard, DeviceGrid — проверка touch targets и адаптивности

**Приоритет:** Low
**Сложность:** Низкая (проверка + мелкие правки)

## Страницы для проверки

Эти страницы уже имеют хорошую адаптивность. Нужна **проверка** и мелкие правки если обнаружатся проблемы.

### Формы (8 страниц)

| Страница | Файл | Что проверить |
|----------|------|---------------|
| LoginPage | `LoginPage.tsx` | Input type=email, кнопки ≥44px, OAuth кнопка full-width на mobile |
| UserCreatePage | `UserCreatePage.tsx` | Форма `max-w-2xl`, поля full-width |
| RoleCreatePage | `RoleCreatePage.tsx` | Чекбоксы пермишенов: touch target |
| DeviceTokenCreatePage | `DeviceTokenCreatePage.tsx` | Форма компактная |
| OnboardingPage | `OnboardingPage.tsx` | Wizard: шаги + форма |
| SettingsPage | `SettingsPage.tsx` | Простая форма |
| ProfilePage | `ProfilePage.tsx` | Данные профиля |
| WebhookSettingsPage | `WebhookSettingsPage.tsx` | URL + secret inputs |

**Общее правило:** `input { font-size: 16px }` — предотвращает auto-zoom на iOS Safari.

### DashboardPage

- Метрики: `grid-cols-1 sm:grid-cols-2 lg:grid-cols-4` ✓
- Графики Recharts: `<ResponsiveContainer>` ✓
- **Проверить:** tooltip графиков не выходит за границы на мобильных

### DeviceGridPage

- Grid: `grid-cols-2 sm:grid-cols-4 lg:grid-cols-6 xl:grid-cols-8` ✓
- **Проверить:** context menu (правый клик) — на мобильных должен работать через long-press или кнопку "..."

### DownloadPage

- Карточки: `grid-cols-1 md:grid-cols-2 lg:grid-cols-3` ✓
- Инструкции: full-width ✓
- **Проверить:** кнопка "Скачать" touch target
