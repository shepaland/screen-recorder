# Фаза 5: Трекинг активности пользователя

## Цель

Реализовать сбор и отправку фокус-интервалов (активное окно, браузер, домен), событий ввода (мышь, клавиатура) и аудит-событий (блокировка экрана, логин/логаут).

## Зависимости

- Фаза 1 (HTTP-клиент, аутентификация)

## Задачи

1. Реализовать `WindowTracker` — определение активного окна, процесса, заголовка
2. Реализовать `BrowserDomainParser` — извлечение домена из заголовка браузера
3. Реализовать `IdleDetector` — определение idle через время последнего события ввода
4. Реализовать `InputEventTracker` — мышь, клавиатура, скролл, буфер обмена
5. Реализовать `AuditEventTracker` — системные события (блокировка, логин)
6. Реализовать `FocusIntervalSink` — батчинг и отправка интервалов
7. Реализовать `InputEventSink` — батчинг и отправка событий ввода
8. Реализовать `AuditEventSink` — батчинг и отправка аудит-событий
9. Offline-буфер: SQLite для хранения непереданных событий

## API-эндпоинты

```

```

**Заголовки:**

### 5.3 Аудит-события

```
POST /api/ingest/v1/audit-events
```

**Интервал отправки:** каждые 30 секунд или при накоплении 100 событий.

**Request:**
```json
{
  "device_id": "...",
  "username": "user",
  "events": [
    {
      "id": "evt-uuid",
      "event_type": "SESSION_START",
      "event_ts": "2026-03-17T10:30:00Z",
      "session_id": "sess-uuid",
      "details": {}
    }
  ]
}
```

**Допустимые `event_type`:**
- `SESSION_START` / `SESSION_END` — начало/конец записи
- `LOGON` / `LOGOFF` — вход/выход пользователя macOS
- `LOCK_SCREEN` / `UNLOCK_SCREEN` — блокировка/разблокировка экрана
- `PROCESS_START` / `PROCESS_STOP` — запуск/завершение приложения
- `INPUT_ACTIVITY` — активность ввода

**Response (200):**
```json
{
  "accepted": 10,
  "duplicates": 0,
  "correlation_id": "corr-uuid"
}
```

## macOS-специфика

### Определение активного окна

```swift
// Активное приложение
let app = NSWorkspace.shared.frontmostApplication
let processName = app?.localizedName  // "Google Chrome"
let bundleId = app?.bundleIdentifier  // "com.google.Chrome"

// Заголовок окна (Accessibility API)
let appElement = AXUIElementCreateApplication(app!.processIdentifier)
var titleValue: AnyObject?
AXUIElementCopyAttributeValue(appElement, kAXFocusedWindowAttribute as CFString, &titleValue)
let window = titleValue as! AXUIElement
var title: AnyObject?
AXUIElementCopyAttributeValue(window, kAXTitleAttribute as CFString, &title)
let windowTitle = title as? String  // "GitHub - screen-recorder"
```

### Геометрия окна

```swift
let windowList = CGWindowListCopyWindowInfo([.optionOnScreenOnly], kCGNullWindowID) as? [[String: Any]]
// Фильтровать по ownerPID == app.processIdentifier
// kCGWindowBounds → { X, Y, Width, Height }
```

**Извлечение домена из заголовка:**
Большинство браузеров показывают домен/URL в заголовке окна. Парсить последний компонент после " - " или " — ".

### Idle-детекция

```swift
let idleTime = CGEventSource.secondsSinceLastEventType(
    .combinedSessionState,
    eventType: .mouseMoved  // или CGEventType(rawValue: ~0)! для любого события
)
let isIdle = idleTime > 300  // 5 минут
```

### События ввода

```swift
// Глобальный монитор мыши
NSEvent.addGlobalMonitorForEvents(matching: [.leftMouseDown, .rightMouseDown]) { event in
    // event.locationInWindow, event.clickCount
}

// Клавиатура — только счётчик (НЕ кейлогер!)
NSEvent.addGlobalMonitorForEvents(matching: .keyDown) { _ in
    keystrokeCount += 1
}

// Скролл
NSEvent.addGlobalMonitorForEvents(matching: .scrollWheel) { event in
    // event.scrollingDeltaY
}

// Буфер обмена
Timer.scheduledTimer(withTimeInterval: 1.0, repeats: true) { _ in
    let currentCount = NSPasteboard.general.changeCount
    if currentCount != lastChangeCount { /* clipboard changed */ }
}
```

### Системные события

```swift
// Блокировка экрана
DistributedNotificationCenter.default().addObserver(
    forName: NSNotification.Name("com.apple.screenIsLocked"), ...)
DistributedNotificationCenter.default().addObserver(
    forName: NSNotification.Name("com.apple.screenIsUnlocked"), ...)

// Запуск/завершение приложений
NSWorkspace.shared.notificationCenter.addObserver(
    forName: NSWorkspace.didLaunchApplicationNotification, ...)
NSWorkspace.shared.notificationCenter.addObserver(
    forName: NSWorkspace.didTerminateApplicationNotification, ...)

// Логин/логаут
NSWorkspace.shared.notificationCenter.addObserver(
    forName: NSWorkspace.sessionDidBecomeActiveNotification, ...)
NSWorkspace.shared.notificationCenter.addObserver(
    forName: NSWorkspace.sessionDidResignActiveNotification, ...)
```

### Разрешения

- **Accessibility** — обязательно для AXUIElement (заголовки окон, UI-элементы)
- Проверка: `AXIsProcessTrusted()`
- Промпт: через `AXIsProcessTrustedWithOptions([kAXTrustedCheckOptionPrompt: true])`

## Критерии приёмки

1. Фокус-интервалы корректно определяют активное окно, процесс, заголовок
2. Для браузеров — определяется имя браузера и домен
3. Idle-детекция работает (is_idle=true после 5 минут бездействия)
4. Геометрия окна (x, y, width, height) передаётся корректно
5. События ввода: счётчик нажатий, клики мыши, скролл
6. Аудит-события: блокировка/разблокировка экрана, запуск/завершение приложений
7. Батчинг каждые 30с, offline-буфер при потере связи
8. Дедупликация по id (UUID) на стороне сервера — агент генерирует уникальные id
