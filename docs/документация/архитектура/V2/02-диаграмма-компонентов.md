## Изменения V1 → V2

| Компонент | V1 | V2 |
|-----------|----|----|
| Количество сервисов | 4 (auth, cp, ingest, web) + MinIO | **9** (+kafka, opensearch, search, playback) |
| confirm() path | Sync DB write → 200 OK | **Kafka publish → 202 Accepted → async DB write** |
| Segment write | ingest-gw → PostgreSQL (sync) | **Kafka → segment-writer consumer → PostgreSQL (batch)** |
| Поиск | Нет | **OpenSearch + search-service (fulltext, tenant-isolated)** |
| Воспроизведение | Нет | **HLS M3U8 + playback-service + hls.js** |
| Webhooks | Нет | **CRUD + Kafka → webhook-dispatcher → HTTP POST + HMAC** |
| Messaging | Нет (планировался NATS) | **Apache Kafka 3.7.0 KRaft (5 topics)** |
| Индексация | Нет | **OpenSearch 2.12.0 (monthly indices, ~3200+ docs)** |
| Мониторинг Kafka | Нет | **Zabbix: 11 метрик, 3 триггера, dashboard "Кадеро"** |
| Network Policies | 12 | **21** (+ search, playback, opensearch, kafka) |
| Frontend страницы | 15 | **18** (+search, playback, webhooks) |
