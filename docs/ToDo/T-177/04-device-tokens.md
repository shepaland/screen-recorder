# T-177.4: DeviceTokensListPage — скрытие колонок на мобильных

**Приоритет:** High
**Сложность:** Средняя (~10 строк)
**Файл:** `web-dashboard/src/pages/DeviceTokensListPage.tsx`

---

## Проблема

Таблица с `min-w-[900px]`. На 375px — горизонтальный скролл. Пользователю приходится скроллить вправо чтобы увидеть действия.

## Исправление

Скрыть второстепенные колонки на мобильных:

| Колонка | Mobile (< sm) | Tablet (sm-lg) | Desktop (lg+) |
|---------|--------------|----------------|---------------|
| Название | Видна | Видна | Видна |
| Токен | Видна (обрезанный) | Видна | Видна |
| Устройств | Видна | Видна | Видна |
| Создан | **Скрыта** | Видна | Видна |
| Последнее использование | **Скрыта** | **Скрыта** | Видна |
| Действия | Видна | Видна | Видна |

```tsx
// Колонка "Создан":
<th className="hidden sm:table-cell">Создан</th>
<td className="hidden sm:table-cell">{token.created_ts}</td>

// Колонка "Последнее использование":
<th className="hidden lg:table-cell">Последнее использование</th>
<td className="hidden lg:table-cell">{token.last_used_ts}</td>
```

Убрать `min-w-[900px]` — таблица будет нормально масштабироваться.
