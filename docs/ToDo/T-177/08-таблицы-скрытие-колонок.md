## Паттерн реализации

```tsx
// Для каждой скрываемой колонки:
<th className="hidden sm:table-cell">Колонка</th>
<td className="hidden sm:table-cell">{value}</td>

// Для колонок скрытых до lg:
<th className="hidden lg:table-cell">Колонка</th>
<td className="hidden lg:table-cell">{value}</td>
```

Если используется компонент DataTable — добавить prop `responsiveHidden`:
```tsx
columns={[
  { key: 'name', title: 'Название' },
  { key: 'slug', title: 'Slug', className: 'hidden sm:table-cell' },
  { key: 'created', title: 'Создан', className: 'hidden lg:table-cell' },
]}
```
