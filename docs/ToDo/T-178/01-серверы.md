# T-178.1: Подготовка серверов — ОС, сеть, firewall, SSH

**Блокирует:** Все остальные задачи

## Серверы

| Имя | Роль | vCPU | RAM | Системный диск | Доп. диск | IP |
|-----|------|------|-----|---------------|-----------|-----|
| **app1** | k3s master, PostgreSQL primary, MinIO, Kafka, OpenSearch | 8 | 16 GB | 150 GB HDD | 2 TB HDD (MinIO) | приватный |
| **app2** | k3s worker, PostgreSQL replica, MinIO, Kafka, OpenSearch | 8 | 16 GB | 150 GB HDD | 2 TB HDD (MinIO) | приватный |
| **lb** | nginx reverse proxy, SSL termination | 2-4 | 4-8 GB | 50 GB | — | **публичный** |

## Шаги

### 1. Заказ серверов
- [ ] Заказать app1, app2, lb у хостинг-провайдера
- [ ] Убедиться что app1 и app2 в **одной приватной сети** (VLAN / VPC)
- [ ] lb имеет публичный IP
- [ ] Записать IP-адреса всех серверов

### 2. ОС
- [ ] Установить Ubuntu 22.04 LTS на все серверы
- [ ] Обновить: `apt update && apt upgrade -y`
- [ ] Установить базовые пакеты: `apt install -y curl wget htop iotop net-tools mc`
- [ ] Настроить timezone: `timedatectl set-timezone Europe/Moscow`
- [ ] Настроить hostname: `hostnamectl set-hostname app1` / `app2` / `lb`

### 3. SSH
- [ ] Сгенерировать SSH-ключ на рабочей машине: `ssh-keygen -t ed25519`
- [ ] Скопировать на серверы: `ssh-copy-id user@app1`
- [ ] Отключить парольную аутентификацию: `PasswordAuthentication no` в `/etc/ssh/sshd_config`
- [ ] Перезапустить sshd: `systemctl restart sshd`

### 4. Firewall (UFW)

**app1 и app2:**
```bash
ufw default deny incoming
ufw default allow outgoing
ufw allow from <lb_ip> to any port 443        # HTTPS от LB
ufw allow from <lb_ip> to any port 6443       # k3s API от LB (опционально)
ufw allow from <app1_ip> to any               # Весь трафик от app1
ufw allow from <app2_ip> to any               # Весь трафик от app2
ufw allow from <admin_ip> to any port 22      # SSH от админа
ufw enable
```

**lb:**
```bash
ufw default deny incoming
ufw default allow outgoing
ufw allow 80/tcp                               # HTTP → redirect 443
ufw allow 443/tcp                              # HTTPS
ufw allow from <admin_ip> to any port 22       # SSH
ufw enable
```

### 5. Приватная сеть
- [ ] Проверить связность: `ping app2` с app1
- [ ] Добавить в `/etc/hosts` на всех серверах:
```
<app1_private_ip>  app1
<app2_private_ip>  app2
<lb_private_ip>    lb
```

### 6. Дополнительный диск (MinIO)
- [ ] На app1 и app2: подключить 2 TB HDD
- [ ] Разметить и смонтировать:
```bash
mkfs.ext4 /dev/sdb
mkdir -p /mnt/s3-data
mount /dev/sdb /mnt/s3-data
echo '/dev/sdb /mnt/s3-data ext4 defaults 0 2' >> /etc/fstab
```
- [ ] Проверить: `df -h /mnt/s3-data`
