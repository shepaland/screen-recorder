## Шаги

### 1. node_exporter на всех серверах

```bash
apt install -y prometheus-node-exporter
systemctl enable prometheus-node-exporter
```

### 2. Prometheus (k8s deployment)

```yaml
# prometheus-config ConfigMap с targets:
scrape_configs:
  - job_name: node
    static_configs:
      - targets: ['app1:9100', 'app2:9100', 'lb:9100']
  - job_name: k8s-pods
    kubernetes_sd_configs:
      - role: pod
  - job_name: postgresql
    static_configs:
      - targets: ['app1:9187']  # postgres_exporter
  - job_name: kafka
    static_configs:
      - targets: ['app1:9308', 'app2:9308']  # kafka JMX exporter
  - job_name: minio
    static_configs:
      - targets: ['app1:9000']
    metrics_path: /minio/v2/metrics/cluster
```

### 3. Grafana дашборды

- Node Exporter Full (ID: 1860)
- PostgreSQL (ID: 9628)
- Kafka Overview
- MinIO Dashboard
- Кастомный: Кадеро (устройства, сегменты, heartbeats)

### 4. Алерты

```yaml
groups:
  - name: kadero-alerts
    rules:
      - alert: ServiceDown
        expr: up{job="k8s-pods"} == 0
        for: 2m
        labels: { severity: critical }
        annotations: { summary: "{{ $labels.pod }} is down" }

      - alert: PostgresReplicaLag
        expr: pg_replication_lag > 30
        for: 5m
        labels: { severity: warning }

      - alert: DiskUsageHigh
        expr: (1 - node_filesystem_avail_bytes{mountpoint="/"} / node_filesystem_size_bytes{mountpoint="/"}) > 0.8
        for: 5m
        labels: { severity: warning }

      - alert: MinIODiskHigh
        expr: (1 - node_filesystem_avail_bytes{mountpoint="/mnt/s3-data"} / node_filesystem_size_bytes{mountpoint="/mnt/s3-data"}) > 0.8
        for: 5m
        labels: { severity: warning }

      - alert: SSLCertExpiry
        expr: probe_ssl_earliest_cert_expiry - time() < 14*24*3600
        for: 1h
        labels: { severity: warning }
```

### 5. Оповещения

AlertManager → Telegram bot:
```yaml
receivers:
  - name: telegram
    telegram_configs:
      - bot_token: '<bot_token>'
        chat_id: <chat_id>

```
